# 3DOF-Robotic-Arm-Controller

（开发中）基于张大头步进电机的RRR构型三自由度机械臂实现 上位机部分

目前实现了串口通信协议下的正逆运动学解算

三臂长分别为：

```c++
L1 = 124.75
L2 = 190
L3 = 197
```
串口命令：

| 命令前缀  | 参数格式         | 描述                                        |
| --------- | ---------------- | ------------------------------------------- | 
| `ik`    | `r theta h`               | 运动学逆解                          | 
| `fk`   | `theta1 theta2 theta3`               | 运动学正解             |
